# Security Basics Lab 🔐

Este repositório reúne estudos, anotações e scripts feitos durante minha jornada de transição para a área de Segurança da Informação. Aqui você encontrará conteúdos voltados para fundamentos em redes, Linux, LGPD, além de pequenos projetos e exercícios práticos.

## 📁 Estrutura
- `redes/`: Conceitos básicos de protocolos, modelos OSI e TCP/IP.
- `linux/`: Comandos úteis e boas práticas no terminal.
- `scripts/`: Scripts simples em Bash e Python aplicados à segurança.
- `vulnerabilidades/`: Relatórios de labs (ex: TryHackMe) com identificação e mitigação.
- `lgpd/`: Princípios e direitos da LGPD estudados.
